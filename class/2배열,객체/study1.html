배열 - array

연관성있는 데이터들을 묶어서 관리하기위한 가공데이터형태

배열안의 콤마로 구분되어있는 데이터 -> 배열의 원소

배열의 원소는 입력한 순서대로 index가 생성된다. 0부터 카운팅 된다.

배열의 원소는 배열의 index순번을 통해 읽을 수 있다.

객체

연관성있는 데이터들을 묶어서 관리하기위한 가공데이터형태 - 각각의 데이터에 이름을 달아주는 형식으로 구성.

객체는 콤마로 구분하여 프로퍼티들이 나열되어있는 형태

객체 안에 들어가있는 프로퍼티는 입력한 순서와는 아무 관계가 없다(순번이 만들어지지 않는다.)
