자바스크립트의 특징

위에서 아래로 해석하는 흐름을 갖고있다. 좌에서 우로 해석하는 흐름을 갖고있다.

기본적으로 좌우로 해석하지만 괄호먼저 해석.

자바스크립트는 언제나 계산을 한다.

;(세미콜론)은 실행이 끝났다. 라는 것을 명시적으로 알려주는 문자.(스크립트 코드안에서 엔터를 쳐서 줄바꿈을 하면 암묵적으로 ;을 입력한다.)

데이터형태

6가지의 데이터 형태를 갖고 있음.

  • string - 문자
  • number - 숫자
  • boolean - 참거짓
  • array - 배열
  • object - 객체
  • function - 함수

string

쌍따옴표,홑따옴표,백틱으로 마킹하는 데이터

문자데이터끼리 덧셈이 가능 -> 서로 연결이되고 문자데이터로 계산됨.

number

마킹없이 숫자로만 구성된 데이터

사칙연산(+,*,/,-)+나머지연산(%)이 가능

정확한 계산을 해야하는 숫자데이터는 관용적표현을 쓸 수 없음(예:010)

문자와 숫자를 더하면 이어지면서 문자가된다.(계산X)

boolean

참거짓데이터

비교에 의해서 참거짓이 계산됨

부등호,등호(연산문자)

  • 크다(>)
  • 작다(<)
  • 크거나같다(>=)
  • 작거나같다(<=)
  • 같다(==) 데이터형태를 따지지 않은 결과-느슨한 비교
  • 같다(===) 데이터형태까지 깐깐하게 따진 결과-타이트한 비교

존재의 여부에 의해서 참거짓이 간주됨

무엇무엇이 있다 없다라는 기준으로 결과를 간주함

false로 간주되는 표현

  • ""빈문자데이터는 false로 간주된다.
  • 숫자데이터0은 false 간주된다.
  • 값을 입력 하지않은 변수도 false로 간주된다.
  • undefined(의도되지않은-비어있는값)/null(의도된-비어있는값)/NaN(Not a Number)
